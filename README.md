# GramSathi AI 🇮🇳
> **Find Opportunities. Stay Protected. Empower Communities.**

GramSathi AI is a premium, AI-powered Opportunity Discovery and Cyber Safety Platform designed for rural and semi-urban Indian communities. It acts as a trusted digital companion, empowering citizens to discover eligible welfare schemes, scholarships, and agricultural subsidies while protecting them from online fraud, phishing, spoof portals, and fake loans.

---

## 🌟 Key Features

1. **Opportunity Discovery Engine**: Calculates a citizen's eligibility score dynamically using demographic indicators (age, state, income, landholdings, education) and displays eligible central and state schemes.
2. **Opportunity Radar**: An interactive SVG-based radar chart illustrating matches across Agriculture, Education, Healthcare, Employment, Entrepreneurship, and Finance.
3. **Future Impact Simulator**: A projection tool showing how welfare programs can step-up monthly income, rendered on a dynamic SVG line chart.
4. **SchemeShield AI Scan**: A simulated cybersecurity scanner that evaluates domains, screenshots, and SMS text messages for phishing indicators, outputting a detailed trust report and risk score.
5. **Multilingual AI Companion**: An interactive chatbot supporting English, Hindi, Marathi, Tamil, Telugu, and Kannada with built-in voice synthesis (Text-to-Speech) using the Web Speech API.
6. **Village Intelligence Dashboard**: A dashboard reflecting village-level missed subsidies, active cyber threats, and digital inclusion scores, filterable by state, district, and village.

## 🚀 Live Deployment Instructions (GitHub Pages)

You can host this website online for free using **GitHub Pages** in under 5 minutes:

### Step 1: Create a GitHub Repository
1. Log in to [GitHub](https://github.com).
2. Click **New** to create a new repository.
3. Name your repository (e.g., `gramsathi-ai`).
4. Set the visibility to **Public**.
5. Click **Create repository** (do not add a README, license, or gitignore yet).

### Step 2: Push the Files to GitHub
Open your terminal (PowerShell, Bash, or Git Bash) inside the project folder:

```bash
# Initialize a local Git repository
git init

# Add all files to staging
git add .

# Commit changes
git commit -m "Initial commit: GramSathi AI Platform"

# Rename branch to main
git branch -M main

# Link to your remote GitHub repository (replace with your actual GitHub URL)
git remote add origin https://github.com/YOUR_USERNAME/gramsathi-ai.git

# Push files
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub.
2. Click on the **Settings** tab.
3. In the left-hand sidebar under "Code and automation", click **Pages**.
4. Under **Build and deployment** -> **Source**, select **Deploy from a branch**.
5. Under **Branch**, choose `main` and select `/ (root)` folder. Click **Save**.
6. Wait 1–2 minutes. GitHub will generate a live URL for your project (e.g., `https://YOUR_USERNAME.github.io/gramsathi-ai/`).
